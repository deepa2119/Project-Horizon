# Project Horizon

A workspace for knowledge curation, research, experiments, automation, and project management.

## Directory Structure

- **Knowledge/**: Curated notes and documentation.
  - **AI/**: Artificial Intelligence & Machine Learning notes.
  - **Crypto/**: Cryptography, security, and web3 materials.
  - **DevOps/**: Infrastructure, CI/CD, and orchestration.
  - **Linux/**: OS-level configurations, scripts, and administration.
  - **Hardware/**: Electronics, physical computing, and system specs.
- **Experiments/**: Scratchpads, proofs of concept, and prototype testing.
- **Projects/**: Main active development projects.
- **Benchmarks/**: Performance testing results and comparison studies.
- **Journal/**: Daily/weekly work logs and reflections.
- **Resources/**: References, external assets, and utility files.
- **Automation/**: Cron jobs, scheduled tasks, and automated workflows.
- **Scripts/**: Utility scripts and helper tools.
- **Archive/**: Deprecated or inactive work.
